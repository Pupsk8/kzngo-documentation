Рекламный блок

Контроллер - 
1. Добавление 
2. Удаление(Деактивация) - выставление статуса 0(полного удаления не существует)
3. Редактирование

На index показаны автивные блоки каждого из 4 разделов
Места -новые
Места - популярные
С детьми - события
С детьми - места


###
Таблица
id
idNode - айди места(мероприятия, для детей, новости и т.д.)
type - STATIC_ID_OF_MATERIALS (тип)
section - секция(пример: место новое или популярное)
dateBegin - начало публикации
dateEnd - конец публикации
inset - присутствует ли в табе
status - активный, не активный

существующие типы(type)
==== Места =====
0 - Новые
1 - Популярные
==== События =====
2 - Популярное
==== С детьми =====
3 - события
4 - места


существующие статусы(status)
0 - скрыт
1 - активен

##########################################################

STATIC_ID_OF_MATERIALS

0 - места
1 - события
2 - кино 
3 - с детьми события
4 - с детьми места
5 - новости
6 - интересное
